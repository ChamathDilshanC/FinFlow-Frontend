# FinFlow Frontend (Mobile App)

FinFlow mobile client built with Expo and React Native. This app provides a polished personal finance experience for tracking subscriptions, transactions, budgets, and spending insights.

## App Features

- Secure authentication and session handling
- Transaction and subscription management
- Category-based budgeting workflows
- Dashboard metrics and insights
- Export, print, and share support

## Local Setup

```bash
cd frontend
npm install
npm run start
```

## Available Scripts

- `npm run start` - Start Expo (LAN)
- `npm run start:default` - Start Expo default mode
- `npm run start:tunnel` - Start Expo tunnel mode
- `npm run android` - Open Android via Expo
- `npm run ios` - Open iOS via Expo
- `npm run web` - Open web preview
